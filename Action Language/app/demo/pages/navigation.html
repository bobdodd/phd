About This Pattern

Navigation must use semantic HTML, provide skip links, and support keyboard access.

  • WCAG Criteria: 2.1.1 (Keyboard), 2.4.1 (Bypass Blocks), 4.1.2 (Name, Role, Value)
  • Required ARIA: role="navigation" (if not using nav), aria-current for active page

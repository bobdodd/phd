ARIA Radiogroup Requirements

Essential Requirements for Accessible Radiogroups

  • role="radiogroup" on the container element
  • role="radio" on each radio option element
  • aria-checked="true" on selected option, aria-checked="false" on others
  • aria-labelledby or aria-label on the radiogroup to describe the group
  • Arrow keys (Up/Down or Left/Right) navigate between options with roving tabindex
  • Space key selects the currently focused option
  • Only one option can have tabindex="0" at a time (roving tabindex pattern)
  • Unselected, non-focused options have tabindex="-1"

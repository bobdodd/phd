What is a Meter?

A meter represents a scalar measurement within a known range, or a fractional value. Unlike progress bars (which show task completion), meters display measurements like disk usage, battery levels, ratings, and temperature.

Accessibility Requirements:

  • <meter> element (native HTML) OR role="meter" with ARIA attributes
  • aria-valuenow - Current value of the meter
  • aria-valuemin - Minimum value (usually 0)
  • aria-valuemax - Maximum value
  • aria-label or aria-labelledby - Accessible name
  • aria-valuetext (optional) - Human-readable text representation
  • Semantic ranges - low/optimum/high with visual indicators

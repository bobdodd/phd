Hierarchical tree with 4-way arrow navigation

Very High Complexity WCAG 2.1.1, 4.1.2

Complete Tree Pattern Requires:

  • role="tree" on container
  • role="treeitem" on each node
  • aria-expanded on expandable nodes
  • aria-level indicating depth
  • aria-setsize and aria-posinset for position
  • Arrow Up/Down: Navigate siblings
  • Arrow Right: Expand node or move to first child
  • Arrow Left: Collapse node or move to parent
  • Enter/Space: Activate or expand/collapse
  • Home/End: Jump to first/last visible node

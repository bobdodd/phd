About Grid Pattern

A grid is a composite widget containing one or more rows of cells. Each cell can contain text, images, links, buttons, or other widgets. Grids support 2D keyboard navigation similar to spreadsheet applications.

  • Use Case: Data tables with interactive cells, spreadsheets, editable tables, complex data displays
  • Key Features: Arrow key navigation, column sorting, cell selection, keyboard shortcuts
  • Accessibility: Full keyboard control, screen reader support, focus management

Common Accessibility Problems

This implementation demonstrates common mistakes that make data grids inaccessible. Paradise analyzer will detect 8 critical issues.

⚠️ Paradise will detect these issues:

  • Missing role="grid" on table container
  • Missing role="row" on table rows
  • Missing role="columnheader" on header cells
  • Missing role="gridcell" on data cells
  • No keyboard navigation support (arrow keys)
  • Missing aria-sort on sortable columns
  • No focus management (tabindex not implemented)
  • Missing aria-label for grid identification

Selectable list with keyboard navigation

High Complexity WCAG 2.1.1, 4.1.2

Complete Listbox Pattern Requires:

  • role="listbox" on container
  • role="option" on each item
  • aria-selected on options (true/false)
  • aria-multiselectable if multiple selection
  • Arrow keys: Navigate options
  • Space: Toggle selection (multi-select)
  • Home/End: Jump to first/last
  • Type-ahead: Jump to matching option

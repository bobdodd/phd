Autocomplete input with arrow navigation and selection

Very High Complexity WCAG 2.1.1, 4.1.2

Complete Combobox Pattern Requires:

  • role="combobox" on input element
  • aria-expanded on input (true/false)
  • aria-controls pointing to listbox
  • aria-activedescendant for virtual focus
  • role="listbox" on dropdown
  • role="option" on each item
  • Arrow keys: Navigate options
  • Enter: Select option
  • Escape: Close dropdown
  • Type-ahead: Filter options by typing

Medium Complexity WCAG 2.1.1, 4.1.2

Complete Toolbar Pattern Requires:

  • role="toolbar" on container
  • aria-label on toolbar for identification
  • Button elements or role="button"
  • Roving tabindex: Only one item at tabindex="0"
  • Arrow Left/Right: Navigate buttons
  • Home/End: Jump to first/last
  • Enter/Space: Activate focused button

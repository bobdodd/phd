Single Letter Shortcut Test Cases

  • 's' key on document without modifier (line ~12)
  • 'h' key on document without modifier (line ~19)
  • keyCode 65 ('A') without modifier (line ~47)
  • 'd' and 'e' keys without modifier (lines ~54-58)

Expected Good Patterns:

  • Meta+K (has modifier)
